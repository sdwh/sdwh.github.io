關於我

學習資訊科學的歷程跌跌撞撞，參與過學院式教育、基於興趣的買書自學，也曾以OCW的方式進行學習，最後是在四分溪畔旁的辦公室無數個夜晚，感受到一次次解決問題的喜悅，確認志業在此。

書寫部落格的目的很簡單，為了在參與資訊科技的進展過程，達成每年規劃的學習目標，並利用筆記強化學習的成果，同時也易於工作中的檢索。

👨‍💻 目前的工作主要是為了解決 IT 上面各式的問題，純粹的開發反而更多時間是基於自己的興趣，而 Web 是學習一路以來最有興趣的領域，也因此筆記學習的對象大多是 Web 相關的技術，並輔以工作上 IT 相關的技術筆記，因此內容的核心就是紀錄過程中的學習與解決問題的快樂 (arbejdsglæde) 😉

部落格中的文章是基於工作與興趣所做的研究與整理，如果有不當引用或者內容錯誤的話，歡迎各位不吝指正 🙏
