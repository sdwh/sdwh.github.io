Visual Studio Subscriptions

  1. 說明
    1. Visual Studio Subscription Benefits

筆記 Visual Studio 訂閱版本 (Subscription) 與獨立授權 (Standalone)。

說明

訂閱版本 (Subscription) 與獨立授權 (Standalone) 最大的差別就是訂閱版本可以使用最新的版本，但因此價格長期而言會較獨立授權 (Standalone) 昂貴，此外訂閱版本 (Subscription) 提供了一些額外的權益功能 (Benefits)，例如 Azure Credit 以及 Azure DevOps Service，有需要使用的情況下可以節省額外的費用。

Visual Studio Subscription Benefits

使用 Visual Studio 訂閱版包含數項重要的權益功能 (Benefits)

  • Azure DevOps Service
  • Azure DevOps Server User CAL
  • Software for dev/test (Windows & Windows Server)
  • Azure Credit $50 per month
  • LinkedIn Learning 3 Months
  • Pluralsight 3 Months
  • DataCamp 3 Months

詳細可見 Standard subscriptions | Microsoft

其中 Azure DevOps Service 與 Azure DevOps Server User CAL 的價值是每年約 72 USD。

另外還有一種授權模式為 Visual Studio Subscription with GitHub Enterprise, GitHub Enterprise 的價值是每月 21 USD，或每年 231 (19.5 × 12) USD，但目前找不到這種訂閱的計價，但可以推估應該會介於 799 至 1030 USD之間。
