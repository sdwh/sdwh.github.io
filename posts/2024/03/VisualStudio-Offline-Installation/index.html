Visual Studio Offline Installation

  1. 說明

筆記如何製作 Visual Studio 離線安裝檔。

說明

先下載 vs_Community.exe。

儲存在適當的 Drive 後，使用 cmd 切換，並執行以下指令開始下載，可以依照需要的 workload 去調整。

預設的下載路徑會儲存在 c:\localVSlayout 可以另外依照需求去調整。

vs_Community.exe --layout c:\localVSlayout --add Microsoft.VisualStudio.Workload.NetWeb --add Microsoft.VisualStudio.Component.Git --lang zh-TW
