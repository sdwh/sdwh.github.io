Windows-11 安裝不使用 Microsoft Login

  1. 說明

說明如何安裝 Windows 11 的時候，不需要透過 Microsoft Login 來完成整個安裝過程 (Local Machine Account Only) 😀

說明

微軟似乎試圖在建立整個使用環境的生態系，所以在安裝 Windows 11 的時候會引導使用者登入 Microsoft Login。

但有些設備屬於共用性質，不適合登入，僅適合使用 Local Machine Account 來安裝與使用。

在安裝過程在選舉國家與地區的階段，使用 Shift + F10 接著輸入 OOBE\BYPASSNRO 後重新啟動。

並且主動停止連網 (不連線 Wi-Fi 或不插網路線)，或者是再次使用 Shift + F10 輸入 ipconfig /release 來停止網路。

接著在安裝上就可以有選項進行不透過 Microsoft Login 來完成安裝了。

但安裝完成後每次的更新，Windows 11 都會試圖引導使用者登入 Microsoft Login。

可以從「設定」的「帳戶設定」，從「登入選項」找到「每次更新後，使用登入資訊進行登入」並停用，就不會再有擾人的登入提示了 😀
