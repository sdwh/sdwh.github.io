Full Mappings

從 SQL Server DataType 到 .NET Framework 除字串類型相容 String 以及 Char[] 都情況外，都是一對一的轉換；但如果是從 .NET Framework Type 要轉換成 SQL Server DataType 就是多對一的情況，沒有辦法直接轉換。這個問題對於 Code First 來說應該有另外的處理方法？
