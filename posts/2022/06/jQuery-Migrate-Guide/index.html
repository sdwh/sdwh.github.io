李代桃僵

如果需要調整的參考 jQuery 頁面過多，可以直接把連結目標的內容進行調整，搭配 git 使用，在測試過程可以及時恢復。

各版本的 jQuery

live API Deprecated

由原本的 live API：

$('#nav a').live('click', function(){ ... })

調整為 on API 用以取代。

$(document).on('click','#nav a', function({ ... })
