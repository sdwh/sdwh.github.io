SQL Server Computed Column

  1. 說明
  2. 參考資料
  3. 相關連結

說明如何在 SQL Server 使用 Computed Column (計算資料行)，由任意的資料表欄位來計算出新的欄位數值。

說明

使用 T-SQL 加入欄位：

ALTER TABLE [AdventureWorksLT2019].[SalesLT].[Product]
  ADD ComputedColumn AS ([Weight]*[ProductId]) PERSISTED

也可以使用 SSSM GUI 來操作

如果使用 PERSISTED Computed Column 才會儲存在 Page 當中，否則會是動態計算出的數值。
