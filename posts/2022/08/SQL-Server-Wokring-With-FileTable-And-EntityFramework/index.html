SQL Server 搭配 Entity Framework 使用 FileTable

  1. 說明
  2. 參考資料
  3. 相關連結

筆記使用 FileTable 藉由 SQL Server 管理 NTFS 檔案儲存，並且搭配 EntityFramework 提供應用系統調用，解決以往使用者上傳附件保存的問題。

說明

SELECT *
  FROM [Database].[dbo].[FileTable]
  WITH (READCOMMITTEDLOCK)
INSERT INTO [dbo].[FileTable]
  ([name],[file_stream])
  SELECT 'books.jpg', * 
  FROM OPENROWSET(BULK N'D:\images\books.jpg', SINGLE_BLOB) AS FileData
GO

載入檔案至 FileTable
