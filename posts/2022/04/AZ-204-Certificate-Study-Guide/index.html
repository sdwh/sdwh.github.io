說明如何準備 AZ-204 Microsoft Azure Developing Solutions 證照，從測驗包含的範圍、參考學習資源及練習題庫資源。並說明如何實際使用 14 天進行準備，以高分通過測驗並順利取得證照的心得分享。

AZ-204

什麼是 AZ-204

AZ-204 是一項微軟的測驗名稱，全名為 「Exam AZ-204: Developing Solutions for Microsoft Azure」，通過這門測驗可以取得「Microsoft Certified: Azure Developer Associate」證照。

在概念上容易混淆，AZ-204 是測驗的名稱，只要通過 AZ-204 測驗可以取得 Azure Developer Associate 證照，但大家習慣直接稱為 AZ-204 證照，而不去細究兩者的差別（測驗名與證照名）。

Azure Developer Associate 是屬於二星級 Associate 等級的證照，相較於 AZ-900、AI-900、DP-900 以及 SC-900 是一星級的 Fundemental 證照，而 「Microsoft Certified: DevOps Engineer Expert」則是需要通過 AZ-104 或者 AZ-204 測驗，並搭配 AZ-400 測驗，才能夠取得的三星級 Expert 證照。

Azure 證照 等級 條件
Azure Data Fundamentals ⭐ 通過 AZ-900 測驗
Azure Administrator Associate ⭐⭐ 通過 AZ-104 測驗
Azure Developer Associate ⭐⭐ 通過 AZ-204 測驗
DevOps Engineer Expert ⭐⭐⭐ 通過 (AZ-104 or AZ-204) and AZ-400 測驗

我需要考 AZ-204 嗎

Azure 的好，用過都知道。你是否好奇下列這些問題：如何從將網站部署至 IIS，改為發佈至 Azure 上，而發佈至 Azure 上又有那些可以採用的解決方案？

可以採用的解決方案包含了 IaaS、容器化、PaaS 或者是 Serverless 等方式，但什麼樣的方式才是最適合的？又該如何調整開發方式使用 Azure 相關 SDK 進行開發呢？

藉由準備 AZ-204 證照，不僅可以回答上述問題，學習如何使用 Azure 雲端資源進行網站、系統、服務的部署，更可以設計高可用 (High Availability)、低延遲 (Low Latency) 並且能夠因應需求縮放伺服器服務數量 (Auto Scaling) 的架構。打造從雲端觀點出發，面向全球存取需求等級的 「雲端原生應用系統」(Cloud Native Apps ) 。

你也想考 AZ-204 並加入成為開發 Native Cloud Apps 的一員嗎，那就讓我們開始介紹如何準備考試吧 😀

如何準備 AZ-204

從微軟的 AZ-204 測驗介紹頁面，我們可以看到測驗所要評量的學習內容以及配分比例，主要包含下列五個主題：

  • Develop Azure compute solutions (25-30%)
  • Develop for Azure storage (15-20%)
  • Implement Azure security (20-25%)
  • Monitor, troubleshoot, and optimize Azure solutions (15-20%)
  • Connect to and consume Azure services and third-party services (15-20%)

各主題的更詳細介紹，可以參考微軟的 AZ-204 Skills Measured Outline

Develop Azure compute solutions 介紹使用 IaaS 的方式，將系統部署至 Azure，包含使用 VM 以及 Container 的方式；此外介紹了 Paas 的 Azure App Service 以及 Serverless 的 Azure Function，讓開發人員專注在功能的實現，而非底層的伺服器環境。

Develop for Azure storage 介紹如何在 Azure 上進行 Storage 結合系統功能進行保存檔案、圖片、文件等；此外也介紹了微軟的 NoSQL 服務 Cosmos DB 以及其 SDK 的使用方式。

Implement Azure security 介紹如何讓 Azure 的各種 Service 之間透過安全的方式進行存取；說明如何以 Azure 內建的機制進行使用者的驗證與授權，其技術相容於 OAuth 2.0 以及 OpenID。

Monitor, troubleshoot, and optimize Azure solutions 介紹如何監控 Azure 上服務的各種資訊，透過 Azure App Insight 並搭配 Log Analytics 作為所有 Log 的集合入口。

Connect to and consume Azure services and third-party services 介紹開發雲端原生系統所要搭配的 Message Queue, Event Grid, CDN 以及 Redis Cache 技術以及相關的 SDK 使用方式。

學習方式

要正式展開 AZ-204 的學習，分為兩種方式 自學 (Self-Taught) 或者參加為期五天的 訓練課程 (Instructor-led) 。

自學方案可以使用免費的 Microsoft Learn 或者需要訂閱制度的 Pluralsight 或者在 Udemy 一次付費買斷課程，由 Scott Duffy 授課的AZ-204 Developing Solutions for Microsoft Azure Exam Prep，此外在 Youtube 上也有相關的資源可以參考，例如 Youtube Skill Me UP Academy 的AZ-204 Azure Developer - Exam Cram。

上述的資源也可以混合使用，筆者建議先以 Microsoft Learn 開始學習旅程，如果英聽 OK 的話再搭配 Youtube Skill Me UP Academy 的AZ-204 Azure Developer - Exam Cram，在按照學習情況決定是否要付費使用 Pluralsiht 以及 Udemy。

而使用自學方案的話，仍不要錯過訓練課程會介紹的實作課程 Labs，如果對於 Azure 有一定實力的朋友可以自己跟著 Labs 步驟實作，如果有問題也可以參考 Youtube 上 okay azure 示範 Labs 操作的 AZ-204 Labs，Labs 會用到的相關素材可以從 GitHub 進行下載。

此外筆者也整理了在準備證照期間，實作各 Lab 的過程以及心得：

實作 AZ-204 Lab 01 App Services
實作 AZ-204 Lab 02 Azure Functions
實作 AZ-204 Lab 03 Storage SDK
實作 AZ-204 Lab 04 Cosmos DB
實作 AZ-204 Lab 05 Azure Container
實作 AZ-204 Lab 06 Authenticate
實作 AZ-204 Lab 07 Key Vault
實作 AZ-204 Lab 08 API Management
實作 AZ-204 Lab 10 Azure Service Bus Queues
實作 AZ-204 Lab 12 Azure Content Delivery Network, CDN

訓練課程 會得到官方的培訓教材 (電子檔)以及為期一個月的 Azure 資源使用帳戶 (額度 100美金)，辦理機構的選擇可以參加恆逸資訊辦理的 AZ-204 Microsoft Azure開發解決方案 課程，可以選擇實地上或者遠距上課，參加恆逸課程的會提供培訓教材的紙本列印、考前總複習以及原地的測驗機會，測驗的報名會有客服協助，對於初次接觸 Azure 證照考試的朋友相當有幫助。

此外創能資訊也有辦理辦理的 AZ-204 - Developing Solutions for Microsoft Azure 可供參加。

測驗資源

有了充足的準備之後，我們就可以開始準備測驗，但在實戰之前來點模擬考讓自己更熟悉應考的方式，會讓通過測驗的機會大增。

Whizlabs 提供免費但有限的 Practice Tests，如果想要使用完整的測驗也可以考慮付費購買。

Examtopics 也有可以供練習的題目，如果有參加過多人報考的證照場次，會發現很多人在考前都在用 Examtopics 練習，做考前的最後衝刺 😉

此外如果你所服務的公司有加入微軟的 ESI Program 可以使用 Program 所提供的 Practice Test 來進行模擬練習以及仿真測驗，根據筆者實際使用的情形，題目非常接近實際的應考題目，很多在正式測驗中所出現的重點，在 Exam Preparation 的練習當中也有出現，超級棒！

目前 ESI Program 關於 AZ-204 的題目約有 150 題，不僅有標準答案更有詳細的說明，各位朋友千萬不要錯過了。

咦？如何知道公司有沒有加入微軟 ESI Program？直接進入 ESI Program 的網站，輸入服務公司的 Email 後，如果有加入的話，網頁就會引導你完成後續的登入以及綁定動作 😎

小結

從準備到實際完成測驗取得證照，筆者總共花了 14 天，其中 7 天是參加為期五天的訓練課程，完成所有的 AZ-204 Labs，並且根據 Labs 實戰過程學習到的觀念瀏覽 learn.microsoft 深化知識。

參加訓練課程的過程中，講師會用微軟的簡報帶領學員認識 AZ-204 相關的知識，過程中也一邊翻閱微軟提供的電子教材。但沒有參加訓練課程的朋友也不需要擔心，在 Pluralsight 的 Learning Path 教學影片其實和訓練課程非常相近。

完成課程後接著開始從 ESI Program 的 Practice Test 開始練題目，不會的地方就去查電子教材或者 learn.microsoft，練習到最後手感有了，每次做 ESI Program 的 Practice Tests 都有 80 分以上的水準。

而實際的應考，滿分 1000分並且 700 分級格的測試中，也取得了將近 900 的成績 😍

祝福有興趣各位參加 AZ-204 的朋友都能夠順利通過測驗並取得證照。

如果對於筆者參加 AZ-104 Microsoft Azure Administrator 證照準備過程有興趣的朋友，也可以另外參考 Microsoft Azure Administrator (AZ-104) 證照心得。
