SQL Server Template Explorer 使用範本瀏覽器

  1. 說明
  2. 參考資料
  3. 相關連結

滴水穿石深入 SQL Server 的使用，藉由 SQL Server Management Studio 內建的範本瀏覽器，讓資料庫的開發與維護工作變得更為輕鬆 😁

說明

SQL Server Management Studio 內建的範本瀏覽器，包含許多常見的資料庫開發與維護工作所需要使用的 T-SQL，例如新增資料庫或者是新增函式，此外也可以在範本瀏覽器中加入自行設計的範本。當需要範本時，不用額外再查詢或開啟 .sql 檔案，非常方便。

此外內建的範本，可以在搭配指定範本參數值 (Specify Values for Template Parameters)，讓預設的範本按照客製需求調整。

從 檢視 開啟「範本總管」，或者使用快捷鍵 Ctrl + Alt + T

選擇範本後，在查詢編輯中選擇 查詢 頁籤，開啟「指定範本參數的值」，或者使用快捷鍵 Ctrl + Shift + M

可以使用視窗化的方式，調整各種參數😏

而如果自行設計的範本，會儲存於下列路徑：

C:\Users\UserName\AppData\Roaming\Microsoft\SQL Server Management Studio\18.0\Templates\Sql\*.sql
