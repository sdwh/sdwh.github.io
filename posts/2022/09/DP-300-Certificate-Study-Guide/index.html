說明如何準備 DP-300 Microsoft Administering Microsoft Azure SQL Solutions 證照 (原本的名稱為 Administering Relational Databases on Azure )，從測驗包含的範圍、參考學習資源及練習題庫資源。並說明如何實際使用 14 天進行準備並順利取得證照的心得分享。

DP-300

什麼是 DP-300

DP-300 是微軟的測驗名稱，全名為 「Exam DP-300: Administering Microsoft Azure SQL Solutions」，通過這門測驗可以取得「Microsoft Certified: Azure Database Administrator Associate」證照。

在概念上容易混淆，DP-300 是測驗的名稱，只要通過 DP-300 測驗可以取得 Azure Database Administrator Associate 證照，但習慣直接稱為 DP-300 證照。

Azure Database Administrator Associate 是屬於二星級 Associate 等級的證照，同樣屬於二星級的證照還包括通過 AZ-104 測驗的 Azure Administrator Associate；相較於 AZ-900、AI-900、DP-900 以及 SC-900 等測驗則是取得一星級的 Fundemental 證照，而 「Microsoft Certified: DevOps Engineer Expert」則是需要通過 AZ-104 或者 DP-300 測驗，並搭配 AZ-400 測驗，才能夠取得的三星級 Expert 證照。

Azure 證照 等級 條件
Azure Data Fundamentals ⭐ 通過 AZ-900 測驗
Azure Administrator Associate ⭐⭐ 通過 AZ-104 測驗
Azure Database Administrator Associate ⭐⭐ 通過 DP-300 測驗
DevOps Engineer Expert ⭐⭐⭐ 通過 (AZ-104 or AZ-204) and AZ-400 測驗

我需要考 DP-300 嗎

Azure 的好，用過都知道 😉

而你是否好奇下列這些問題：

雲端資料庫系統有幾種使用方式？IaaS, Paas？同樣屬於 PaaS 服務的 SQL Server Serivces，使用 Azure SQL Database 以及 Azure SQL Managed Instance 有什麼差別？

對於我目前地端的資料庫配置相容的程度如何，我們有在使用 SSIS、SSRS 以及 SSAS 到了雲端是否仍然持續支援。

資料庫維運該如何享受在 Cloud 所提供 Backup 以及 High Availability 優勢？

藉由準備 DP-300 證照，不僅可以回答上述問題，學習如何使用 Azure 雲端資源進行資料庫部署，更可以設計高可用 (High Availability)、低延遲 (Low Latency) 並且能夠因應需求縮放伺服器服務規模 (Auto Scaling) 的架構。搭配開發人員從雲端觀點出發，設計面向全球存取需求等級的 「雲端原生應用系統」(Cloud Native Apps ) 。

如何準備 DP-300

從微軟的 DP-300 測驗介紹頁面，我們可以看到測驗所要評量的學習內容以及配分比例，主要包含下列五個主題：

  • Plan and implement data platform resources (20—25%)
  • Implement a secure environment (15—20%)
  • Monitor, configure, and optimize database resources (20—25%)
  • Configure and manage automation of tasks (15—20%)
  • Plan and configure a high availability and disaster recovery (HA/DR) environment (20—25%)

各主題的更詳細介紹，可以參考微軟的 DP-300 Skills Measured Outline

Plan and implement data platform resources 介紹使用 IaaS (SQL Server On Azure Virtual Machines) 以及 PaaS (Azure SQL Database 以及 Azure SQL Managed Instance) 的雲端資料庫解決方案。

Implement a secure environment 介紹雲端資料庫的安全性機制，包含身分的驗證與授權以及各式資料庫加密技術，例如 TDE 以及 Always Encrypted。

Monitor, configure, and optimize database resources 說明如何藉由 DMOs (DMV, DMF) 來監控雲端資料庫的作業效能，並說明各式可以設定的效能優化選項。優化的技巧不限雲端使用，觀念在地端的資料庫環境也是相通。

Configure and manage automation of tasks 說明在 Azure 如何為 Cloud Database 進行自動化作業的安排，包含從地端既有的 Agent Job 到使用 Elastic Job 以及 Azure Automation。

Plan and configure a high availability and disaster recovery (HA/DR) environment 介紹在 Azure 如何為資料庫進行高可用的配置，包含 Auto-Failover Groups 以及 Active-Geo Replicas

學習方式

要正式展開 DP-300 的學習，分為兩種方式 自學 (Self-Taught) 或者參加為期五天的 訓練課程 (Instructor-led) 。

自學方案可以使用免費的 Microsoft Learn 或者在 Udemy 一次付費買斷課程，此外在 Youtube 上也有相關的資源可以參考，例如 Youtube DbSkyLimit 的DP-300 | Azure Database Administrator Associate Certification。

上述的資源也可以混合使用，筆者建議先以 Microsoft Learn 開始學習旅程，並且在學習的過程中主動向官方文件延伸製作筆記，此外按照學習情況決定是否要額外購買 Udemy 課程搭配學習。

使用自學方案的話，仍不要錯過訓練課程會介紹的實作課程 Labs，可以嘗試跟著 Labs 步驟實作，如果有問題也歡迎大家一起 討論，Labs 會用到的相關素材可以從 GitHub 進行下載。

此外筆者也整理了在準備證照期間，實作各 Lab 的過程以及心得：

Azure Learn DP-300 The Hard Way | 實作 DP-300 Lab 01 Azure SQL Databases
Azure Learn DP-300 The Hard Way | 實作 DP-300 Lab 02 Azure SQL Databases
SQL Server Azure SQL Solutions
SQL Server Migrate to Cloud Tools & Solutions
SQL Server 使用 Azure Storage Account 備份資料庫 (Backup Database To URL)

訓練課程 會得到官方的培訓教材 (電子檔)以及為期一個月的 Azure 資源使用帳戶 (額度 200 美金)，辦理機構的選擇可以參加恆逸資訊辦理的 DP-300 微軟Azure SQL Database資料庫管理 課程，可以選擇實地上或者遠距上課，參加恆逸課程的會提供培訓教材的紙本列印、考前總複習以及原場地的測驗資源，測驗的報名會有客服人員協助，對於初次接觸 Azure 證照考試的朋友相當有幫助。

此外創能資訊也有辦理辦理的 DP-300T00 - Administering Microsoft Azure SQL Solutions 可供參加。

測驗資源

有了充足的準備之後，我們就可以開始準備測驗，但在實戰之前來點模擬考讓自己更熟悉應考的方式，會讓通過測驗的機會大增。

Whizlabs 提供免費但有限的 Practice Tests，如果想要使用完整的測驗也可以考慮付費購買。

Examtopics 也有可以供練習的題目，如果有參加過多人報考的證照場次，會發現很多人在考前都在用 Examtopics 練習，做考前的最後衝刺 😉

此外如果你所服務的公司有加入微軟的 ESI Program 可以使用 Program 所提供的 Practice Test 來進行模擬練習以及仿真測驗，根據筆者實際使用的情形，題目非常接近實際的應考題目，很多在正式測驗中所出現的重點，在 Exam Preparation 的練習當中也有出現，超級棒！

目前 ESI Program 關於 DP-300 的題目約有 120 題，不僅有標準答案更有詳細的說明，各位朋友千萬不要錯過了。

咦？如何知道公司有沒有加入微軟 ESI Program？直接進入 ESI Program 的網站，輸入服務公司的 Email 後，如果有加入的話，網頁就會引導你完成後續的登入以及綁定動作 😎

對於 DP-300 的重要考點，千萬不要錯過筆者所編輯的 SQL Server Azure SQL Solutions 筆記 😏

心得回顧

從準備到實際完成測驗取得證照，筆者總共花了 14 天，其中 7 天是採用自學的方式，完成所有 DP-300 Labs，並且根據 Labs 實戰過程學習到的觀念搭配瀏覽 learn.microsoft 深化知識。

完成課程後接著開始從 ESI Program 的 Practice Test 開始練題目，不會的地方就去查電子教材或者 learn.microsoft。

在實際的測驗過程，值得一提的是本次遇到了 Lab 實作題，需要使用 SSMS 以及 Azure Portal 完成所要求的指令，大約有 10 個小 Task 需要完成。而實際的測驗數量上，總共約 44 個題目，以及一個情境題包含 6 個小題，外加一個 Lab 實作題。

祝福有興趣各位參加 DP-300 的朋友都能夠順利通過測驗並取得證照。
