SQL Server 將資料表資料存成 Insert Script 的方式 (How to Save SQL Server Table as Insert Scritp)

  1. 說明

筆記如何將資料表儲存為 Insert Script 的方式，在部分作業情境可以用於備份資料或是易於批次修改資料使用 🤖

說明

首先選擇使用者資料庫，右鍵後依序選擇工作、產生指令碼。


選擇物件選單中，可以挑選要產生的資料表 (Table)或者是檢視表 (View)。


本步驟為關鍵的一步，必須由進階選單，將要編寫的資料類型選擇為結構描述和資料或者僅限資料，才能夠將資料正確生成為 Insert Scripts。
