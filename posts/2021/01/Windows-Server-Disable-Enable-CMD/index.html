筆記如何從 gpedit.msc 啟用 / 停用 命令提示字元 💬

說明

  1. Windows Run : gpedit.msc
  2. 使用者設定 => 系統管理範本 => 系統
  3. 調整「防止存取命令提示字元」

⚠️ Windows 10 Home Edition 無法使用 gpedit.msc ，必須藉由 scripts 來啟用 gpedit.msc
