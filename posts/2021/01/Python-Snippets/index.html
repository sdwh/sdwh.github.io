Python 101 🐍

Text file read & write

讀寫 txt file 的起手式。

def saveTxt(fileName = 'f.txt'):
    with open(fileName, 'w', encoding = 'utf-8') as f:
        f.writelines('text')
def loadTxt(fileName = 'f.txt'):
    with open(fileName, encoding = 'utf-8') as f:
        return f.readlines()
