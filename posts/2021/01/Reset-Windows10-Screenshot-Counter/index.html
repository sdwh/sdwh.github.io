Windows 10 如何重置螢幕截圖的計數 1️⃣2️⃣3️⃣

  1. 說明
  2. 參考資料

Windows 10 可以使用 Windows + PtrScn 快速的進行螢幕截圖，並會自動將截圖計數命名儲存於圖片路徑，本次筆記為說明如何利用登錄編輯程式 (regedit) 來重置計數。

說明

Windows 10 可以使用 Windows + PtrScn 快速的進行螢幕截圖，圖片會自動保存於

C:\users\<使用者名稱>\Pictures\Screenshots

而每次的截圖會自動以「螢幕擷取畫面 (遞增數字).png」的方式命名，如果想要重置數字可以下列登陸編輯的方式做修改：

reset.reg

Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer]
"ScreenshotIndex"=dword:00000001
