關於如何用最短的時間準備 AI-900 並且順利以 940 分通過證照測驗的心得 😀

證照介紹

AI-900 與 AZ-900、DP-900 相同，都是 Azure 角色認證中入門等級的證照。而 AI-900 的考試範圍主要是對於人工智慧 (AI) 基礎概念，以及微軟認知服務(Cognitive Services)的基礎知識。

在2021年1月27日以後的考試版本，是以下列的配分方式作為命題綱要：

描述 AI 工作負載和注意事項 (15-20%)
描述 Azure 上機器學習的基本原理 (30-35%)
描述 Azure 上的電腦視覺工作負載的功能 (15-20%)
描述 Azure 上自然語言處理（NLP）工作負載的功能 (15-20%)
描述 Azure 上的對話式 AI 工作負載的功能 (15-20%)

就學習者的立場而言可以分為兩大區塊：

  1. 微軟上的人工智慧相關服務有那些以及如何使用
    例如 Azure Machine Learning Studio、Computer Vision、Conversational AI 等等服務。
  2. 關於人工智慧的通則性知識
    例如何謂回歸、分群、分類；影片處理、自然語言處理等人工智慧技術概略的技術原理為何等知識。

而實際測驗內容確實如綱要所包含的範圍，對於想要對 AI 有系統性的初步學習者，或者對於微軟 AI 服務有興趣的朋友，可以選擇準備 AI-900 作為入門人工智慧的方式。

詳細的證照介紹與命題大綱細節，可以參考 Microsoft Docs。

準備過程

基本上以 Microsoft Docs 的 Learning Path 做準備，關於 AI-900 總共有五組 Learning Path，分別是：

開始在 Azure 上使用人工智慧
運用 Azure Machine Learning 製作無程式碼的預測模型
探索 Microsoft Azure 中的電腦視覺
探索自然語言處理
探索對話式 AI

主要以內容閱讀、測驗練習為主，實驗環境則省略不做。全部的流程快速瀏覽下來大概花兩個小時，就可以概略的掌握商業化的 AI 服務究竟有那些項目。

此外如果有訂閱 Linkedin Learning 的朋友，可以觀看 Exam Tips: Microsoft Azure AI Fundamentals (AI-900)。這個課程講師會示範操作 Azure 的 AI 相關服務，可以讓沒有實際操作實驗也能更深入的認識 Azure AI 實際使用起來的感覺。

Udemy 上 Azure 相關證照專業講師 Scott Duffy 也有對於 AI-900 提供教學課程 AI-900 Azure AI Fundamentals Exam Prep In One Day，如果想要更全面掌握這項考試的朋友，也可以考慮參加 Scott Duffy 的 Udemy 課程。

微軟有提供所有基礎證照的範例，各位要準備考試的朋友可千萬不要錯過，
參考這些範例考題，對於考試準備有極大的幫助 😍
AI-900 Free Sample Questions

報名測驗

可以由 Pearson Vue 安排測驗日期與地點，地點選擇上包含巨匠電腦、恆逸教育訓練中心等，報名的過程會需要有 Microsoft 帳號來進行。

此外也可以留意微軟的數位通才入門課活動網頁，不定期會辦理免費的考證課程加上免費提供的證照測驗機會。課程包含線上教學，免費測驗則是提供 Coupon Code 可以於 Pearson Vue 中使用兌換免費的測驗機會，此外活動更會安排與測驗同日的考前複習班，由專業的講師進行測驗前總複習，對於證照測驗有很大的幫助。

測驗進行

實際測驗總共有 60 分鐘可以作答，時間相當充足。另外每個人被分配到的題目與題目數量都不相同，我個人是被分配 50 題。

而題目的類型都是單選題與答案拖曳題，可以自由選擇考試的語言，但沒有繁體中文，所以本次是選擇英文測驗，測驗過程不能使用字典等工具，但實際測驗過程所使用的字彙不會太困難。

另外也有簡體中文的測驗可以選擇，對於非常堅持要中文測驗的朋友，或許可以使用此種測驗方式（過程中題目與答案有提供簡體中文與英文對照）。

後續

AI-900 是初階的微軟人工智慧證照，後續可以繼續往 AI-102、DP-100、DP-200、AZ-104 深入學習。

祝福大家都能夠順利通過測驗 😉
