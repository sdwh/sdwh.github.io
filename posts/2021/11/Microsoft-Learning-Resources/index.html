Microsoft Learning Resources 使用微軟學習資源精進之道

  1. learn.microsoft
  2. MCSA

簡介例如利用微軟 Microsoft Learn 進而由 Certification 準備從 Self-Paced Online Learning 以及 Instructor-led 的方式來進行學習，系統性地學習微軟生態系的知識與工具。本次的介紹以 SQL Server Developing, Adminisration & Business Intelligence 為例。

learn.microsoft

以往的 MSDN, Microsoft Software Developer Network 以及相關的文件平台、資源平台現在都已經統整為 learn.microsoft。不論各種微軟的軟體產品，包含作業系統、資料庫、程式語言、應用程式框架等，都可以由 learn.microsoft 找到對應的說明文件以及教學資料。

此外現在微軟的證照體系以 Azure 平台為主，以往的 MCSA, MCSD 已經退役，而在準備 Azure 證照時， learn.microsoft 可以連結到 Microsoft Learn上面有切割為各子單元的學習模組，提供免費學習。

然而 Microsoft Learn 上的學習模組是為 Azure 證照所設計的，對於地端服務所需要的知識建構仍為不足。這個時候就可以透過以往 MCSA、MCSD 認證的課程設計大綱、測驗大綱的方式來安排系統性的學習。MCSA 以及 MCSD 雖然已經退役，但其下所包含的科目仍有提供 Instructor-led 的授課教學，在台灣包含下列的教學單位：

  • Systex (恆逸資訊)
  • Institute for Information Industry (資策會)
  • Great Master Information (巨匠電腦)
  • Trainocate (創能資訊)
  • Certitraining (全域科技)

在微軟生態系當中，恆逸資訊是其中 Instructor-led 課程最為齊全的，也是規模最大的一家教學單位 🤓

MCSA

從 MCSA 我們可以得到該認證相關的課程與對應的測驗，例如 MCSA: SQL 2016 Database Administration 的其中一個考科 70-764: Administering a SQL Database Infrastructure。

其課程 20764C 可以確認課程的大綱 (Course outline)，並以模組 (Module) 下包含數個課程 (Lesson)，每個模組包含一個實驗 (Lab) 的方式進行內容設計。而各課程 (Lesson) 包含不定的步驟性的示範 (Demo)。

藉由 Module 與 Lesson 的標題，可以一窺這門課程的精隨，藉由 Lab 則能驗證實作，加深印象，而各 Demo 則是務求熟稔的技能，會在真實的工作現場上使用得到。而如果沒有參加需要付費的課程，能可以從 Microsoft Companion Content 直接下載 Comapnion Content File 與 Allfiles Excutable file 來學習，但在沒有講師的引導會比較費力就是了。

而如果有報名課程，可以得到 Handbook 是一本詳實的教材，Module 中各 Lesson 的詳細資訊都在教材當中，此外 Lab 的步驟也只會出現在 Handbook 上，無法由 Companion Content File 得知，較為可惜。但不怕辛苦的話還是可以從 Allfiles Excutable file 跟著 Lab Task 一動接著一動來完成。

而在測驗上則有 Skill Measured 可供參考，例如 70-764 Skill Measured

測驗的正式的學習資源則有 Exam Ref 以及 Practice Test：Exam Ref 70-764 Administering a SQL Database Infrastructure with Practice Test

在 Microsoft Press 上有部分的 Exam Ref 提供特定章節試讀，購買的話大約落在 30 美元的價位。
