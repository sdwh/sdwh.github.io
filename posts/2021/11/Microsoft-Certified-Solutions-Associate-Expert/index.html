微軟的生態系培訓證照 Microsoft Certifications (How to train a DBA with Microsoft Solution)

簡介 MCSA / MCSD / MCSE 的古往今來以及目前的 Microsoft Role based 系列證照，以及在 MCAS / MCSE 紛紛退役的情況下，SQL Server DBA / Developer / Windows Server MIS 該如何培訓。

歷史

📚 1994

微軟推出 MCSE, Microsoft Certified Systems Engineer 微軟認證系統工程師認證

📚 2001

推出 MCSA, Microsoft Certified Systems Administrator 微軟認證系統管理師

採單一產品對照單一認證。

📚 2009

因應考古題以及認證與實務落差問題，修改認證制度以區別 IT 人員工作所需技能：

MCTS, Microsoft Certified Technology Specialist 微軟認證技術專員

MCITP, Microsoft Certified IT Professional 微軟認證IT專業人員

🔍MCSE已死 | iThome

📚 2012

證照考試制度再次更新，首先以雲端解決方案 (Cloud Built) 觀點識別認證技能，此外加入重新認證機制，每 2 至 3 年必須重新認證以證照效力。名稱也由 MCTS 與 MCITP 調整回 MCSA 與 MCSE，但已分別由 Administrator 與 Engineer 改為 Associate 與 Expert。

MCSA, Microsoft Certified Solutions Associate

MCSE, Microsoft Certified Solutions Expert

MCSD, Microsoft Certified Solutions Developer

MCSM, Microsoft Certified Solutions Master MCSM 的認證效力為終生

🔍微軟認證大改制，MCSA和MCSE名稱回來了 | iThome

📚 2016

在大 MOOC 時代的對應出現了 MPD, Microsoft Professional Degree，搭配 edx 線上課程平台，可以完成課程並取得證照，並且有一張證照等級金字塔圖，MPD 作為知識深度的最上層。但該證照似乎曇花一現，由原本 Microsoft Professional Program 網站去連結，目前會導向 Microsoft Learn 看來是已經整併了。

🔍微軟全球首推資料科學 (Data Science) 學位認證 | Microsoft

📚 2020-06-30

MCSA、MCSD、MCSE認證退役，全面改以 Role Based 的證照替代，兩者其實在日期上有所重疊，但 Role Based 的開始日期尚須要確認。

Role Based MS Certification 2021 Nov

🔍微軟6月將終止MCSA、MCSD、MCSE認證考試 | iThome

認證

Database

從 2012 年起，關於 SQL Server 的培訓規劃是完成 MCSA: SQL Server 2012/2014，該認證包含三項考試：

考試 課程 說明
70-461 20461 查詢 SQL Server 2014
70-462 20462 管理微軟 SQL Server 2014 資料庫
70-463 20463 執行微軟 SQL Server 2014 資料倉儲

取得 MCSA 後，在完成下列兩項考試，取得 MCSE: Data Platform 認證：

考試 課程 說明
70-464 20464 開發微軟 SQL Server 2014 資料庫
70-465 20465 設計微軟 SQL Server 2014 資料庫解決方案

或完成下列兩項考試，取得 MCSE: Business Intelligence 認證：

考試 課程 說明
70-466 20466 使用 SQL Server 2014 實現資料模型與微軟 SQL Server 報表
70-467 20467 使用 SQL Server 2014 設計商業智慧解決方案

而從 2016 年起，SQL Server MCSA 再根據細分為下列三類：

一、MCSA: SQL 2016 Database Administration

考試 課程 說明

三、MCSA: SQL 2016 Business Intelligence Development

考試 課程 說明
70-767 20767 執行微軟SQL Server 2016資料倉儲
70-768 20768 開發微軟SQL Server 2016資料模型

此外原本的 MCSE: Data Platform 調整為 MCSE: Data Management and Analytics，取得方式如下：

首先必須具備下列任一認證：

  1. MCSA: SQL Server 2012/2014
  2. MCSA: SQL 2016 Database Administration
  3. MCSA: SQL 2016 Database Development
  4. MCSA: SQL 2016 Business Intelligence Development
  5. MCSA: BI Reporting

接著在完成下列測驗中的其中一項測驗，當然這門測驗不能是已具備的 MCSA 所包含的課程：
