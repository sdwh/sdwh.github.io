WiFi 無線延伸的解決方案

筆記如何延伸家庭環境中的 WiFi 訊號，以常見的技術方式以及市面上可以考慮的產品為例。

說明

網路的重要性不言而喻，隨著行動化裝置的普及以及物聯網技術的興起，每個家庭環境中擁有為數可觀的聯網設備，基本的包含手機、平板、筆電，進階的可能包括手錶、監視器、智慧插座、智慧燈炮等等。而在網路環境架構上，通常都是由 Cable Modem 或者是中華電信使用 PPPoE 作為網際網路訊號來源，並利用 Modem 本身提供的基本 WiFi 功能作為無線設備的網路連結通道。

然而 Modem 本身提供的 WiFi 訊號強度有限，在稍受地形阻隔(水泥牆、金屬板)，連線的品質就會大打折扣，讓使用者體驗嚴重下滑。因此自行安裝獨立的 WiFi Router / Access Point 是有必要的。但是更困擾的情境是即使安裝了新的 AP 裝置，可能仍是受限於地形阻隔，造成空間死角沒有辦法順利進行連線。

因此本篇筆記就是要探討這樣的情形，並說明如何在不用牽實體網路線的方式，進行無線的訊號中繼功能。當然，如果環境是允許牽實體網路線，絕對是優先的考量，因為經過無線傳輸訊號的折損在所難免，使用網路線能夠有最少的干擾並保證網路速度的干擾程度最小。

名詞解釋

什麼是 WiFi Router ？

路由器負責連線網際網路，並具有 DHCP、NAT、Firewall 等功能，提供 WiFi 無線連線形式為其功能之一。

什麼是 Access Point, AP ？

AP 唯一任務是提供 WiFi 無線連線形式，並不提供其他路由器所提供的多重功能。

無線中繼技術

Repeater / AP-Client

使用 Repeater 技術，負責中繼的裝置先以 WiFi 的方式無線連回來源 AP，並使用獨立的 SSID 提供 WiFi 服務。也稱為 AP-Client 模式，其中 Client 是指中繼的路由扮演 Client 角色向來源進行連線。此 Client 相當於是中繼設備中的 Lan Port 以無線方式連回來源 AP。

ASUS 的 AP 設備有所謂的 Repeater Mode，TOTOLINK A720R 也有所謂的 無線中繼模式，歸屬於此類使用方式。

支援的設備如 ASUS RT-N12 ($849)、TOTOLINK A720R ($549)

A720R - Repeater

ASUS - Repeater

WISP

此方案稱為萬用型無線延伸解決方案，負責中繼的裝置先以 WiFi 的方式連回來源 AP，並使用獨立的 SSID 提供 WiFi 服務，在設定上與 Repeater 相當相似，兩者最大的差別在於 WISP 相當於是將中繼 AP 的 WAN Port 以無線的方式連向來源 AP，至於效能與穩定度上需要測試方能得知。

正好 A720R 同時提供 WISP Mode以及 Repeater Mode，兩者的實驗結果待測試 🧪

支援的設備如 TOTOLINK A720R ($549)

A720R - WISP

WDS

WDS, Wireless Distribution System，需要來源 AP 與中繼 AP 都支援 WDS 協定才能夠進行設定。在架構上與 AP-Client 模式相同，WDS 作為中繼 AP Lan Port 上專門協定的無線網卡 Client 需要與來源 AP 也支援 WDS 協定者相互溝通。

WDS 的優點在於可以保持單一 SSID，具有漫遊的特性，但必須要來源與中繼都支援 WDS 才能使用，較為限制。

支援的設備如 ASUS RT-AC1200 V2 ($1,499)

Mesh

支援的設備如 NETGEAR EX7700 ($3,699)

華碩的 AiMesh 強調除了在費用上零缺點的設計，保證效能與傳輸覆蓋範圍，需要華碩特定的無線路由設備才能建構此解決方案。

支援的設備如 ASUS RT-AC86U ($3,499)

HomePlug

電力線是利用電線來進行訊號的傳輸，雖然不是無線傳輸技術，但可以做為無法延伸實體網路線時的替代方案。但使用上受限於電線的特性，也必須考量電路的迴路設計，傳輸性能較受影響。

Zyxel合勤 PLA5456 ($2,999)
