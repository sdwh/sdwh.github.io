全國技能競賽

競賽的目的

基於重視技職教育體系，同時增進民眾對於職業技能的瞭解，並從中培訓國手代表國家參加國際技能競賽。

競賽獎項

除了基本的獎金、獎牌外，選手有機會可以技優升學的方式保送大專院校，此外也可以取得最高乙級技術士測驗免試術科的好處。

如果成為國手，更可以有服補充兵 12 天(一般是 4 個月)以及最高甲級技術士測驗免試術科的好處。

參賽方式

競賽分為青少年組(國中生)與青年組(21歲以下，部分類組24歲以下)。

並須先於分區技能競賽取得各職類前五名。而如果在全國技能競賽取得各職類前三名，則可以參加兩年一次的國手選拔賽，各職類選出一名國手代表餐與國際技能競賽。

資訊與通訊技術類組

資訊網路布建

資訊網路布建（Information network cabling），競賽主要內容為資訊網路規劃、光纖結構化布線系統、銅纜結構化布線系統、無線網路系統、智慧家庭（Smart home）、故障排除及日常維護，以及相關量測技術等。

商務軟體設計

類組舊稱為資訊技術(軟體設計)。

根據企業經營或商務活動所需之軟體設計需要，釐清客戶需求範圍，進行系統分析、系統設計、系統開發、系統測試等系統上線軟體設計流程。使用的軟體包含：

  1. SQL Server Database 設計系統所需的資料庫

  2. Microsoft Visio 規劃軟體設計所需的 ER Diagram 及 UML Diagram

  3. C# (.NET Framework) 搭配 SQL Server 設計應用程式、使用者介面與後台報表 (WPF)

  4. Android Studio 設計 Android 平台運作的 App 行動裝置應用程式

  5. 使用 Excel、Word、PowerPoint 設計商務軟體所需的報告、圖表以及簡報

網頁技術

相較於商務軟體設計注重整個軟體開發流程以及與客戶之間的溝通互動，同時使用的技術平台是 Client-Side 與 Mobile App。

網頁技術著組著重在伺服器端與瀏覽器端的網站設計(Web Design)，包括網站企劃、網頁版面、網頁美工、網頁程式、伺服器端程式設計及資料庫程式設計，同時會需要使用圖片、動畫編輯設計多媒體資源，以呈現各式資訊於網站上，並強調使用者的網頁互動性​。

資訊與網路技術

電腦系統、網路電話、物聯網與網路設備的服務安裝、設定、維護。綜合而言就是 MIS 所要負責的工作，從伺服器、網路設備的管理，到可延伸性、高可用性、備援性及安全性等級的提供。

網路設備則著重在安裝與設定交換器、路由器、資安設備與無線網路裝置，並要能夠以文件與圖檔來清楚表達網路系統的規劃與設計。同時也需具備使用自動化工具與撰寫 Script 管理伺服器、網路設備的能力。

雲端運算

使用 Amazon Web Services(AWS) 作為競賽平台，並由 AWS 進行出題目不公開。

網路安全

分析駭客攻擊手法與技巧，保護資訊系統以及電腦網路，防止敏感資料洩漏。同時須具備監控資訊的能力，並在資訊安全件發生時進行鑑識調查。此外需要具有執行滲透測試的能力，駭客的角度檢察系統是否存在弱點。

僅有「商務軟體設計」與「網頁技術」有設立青少年(國中生)組外，其他各類組都只有設立青年組，所以如果有志參加全國技能競賽的國中生朋友，可以考慮以「商務軟體設計」與「網頁技術」類組的方向進行準備。

小結

個人覺得這個技能競賽提倡技能的價值，並提供技職教育體系展現技能價值的舞台以及為適性學習另闢生涯路徑非常棒。此外資訊與通訊技術類組中的測驗題目，許多都是實際資訊職場中所使的，但在職場中似乎大家不再以技能價值的觀點看待自己所擁有的技術。從這個競賽看到青年對於技術的熱情，也能夠砥礪自己對於技術不斷追求。
