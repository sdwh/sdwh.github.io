Visual Studio 程式碼如何自動換行 (Word Wrap)

  1. 說明
    1. 設定快捷鍵
  2. 相關連結

筆記當程式碼太寬的時候，該如 Visual Studio 何設定自動換行讓開發的閱讀更為方便。

說明

繁體中文環境

執行 Ctrl + Q 後，搜尋換行。

英文環境

執行 Ctrl + Q 後，搜尋Wrap

執行後即可以啟用 / 停止 自動換行的模式

設定快捷鍵

預設上 Visual Studio 自動換行沒有快捷鍵，但可以逕行設定，比照 VSCode 設定為 Alt + Z
