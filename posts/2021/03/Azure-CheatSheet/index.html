Azure 指令筆記 (Azure CheatSheet)

  1. 說明
    1. 資源管理

筆記 Azure 管理過程中經常使用的指令。

說明

資源管理

示範如何建立帳戶

az ad user create --display-name userName --password "Password" ^
--user-principal-name userName@domain.onmicrosoft.com ^
--force-change-password-next-login false;

domain 要調整為 Azure AD 所屬的 Domain

示範如何將帳戶設定為 Resource Group 中的 Contributor

az role assignment create --assignee userName@domain.onmicrosoft.com ^
--role "Contributor" ^
--resource-group "RG-Example";

需要為 Subscription 註冊 Provider 才能夠建立特定的資源

az provider register -n "Microsoft.Compute";
