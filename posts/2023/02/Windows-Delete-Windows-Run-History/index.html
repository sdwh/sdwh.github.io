Windows 10 刪除執行(Windows Run) 的歷史紀錄

  1. 說明

說明如何將 Windows 10 的執行 (Windows Run) 歷史紀錄進行刪除。

說明

刪除的方式是透過對 Register 進行操作，使用 cmd 可以簡化操作的步驟：

REG delete HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\RunMRU /VA /F
參數 用途
/f 不需提示，強制刪除
/va 刪除這個機碼之下的所有值
/ve 刪除空白值名稱的值 (預設值)。

同場加映，如果想要刪除 Explorer 網址列的歷史資料，可以使用相似的方式，不同的路徑進行：

REG delete HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\TypedPaths /VA /F

在上述的作業刪除之前，都可以使用 regedit 前往對應的機碼路徑觀察內容，Enjoy it 😎
