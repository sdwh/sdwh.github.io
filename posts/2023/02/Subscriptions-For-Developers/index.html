LINQPad
快速測試 .NET Code 的工具，提供編輯式以及互動式的介面台，方便測試 Code 以及 Linq
Resharper
Visual Studio Plugins，用於提高開發人員的效率和生產力，提供了強大的代碼分析和優化、代碼自動完成及快速修復等功能。
Rider
同屬 Resharper 開發公司的 IDE 開發環境，用於支持 .NET、.NET Core 及 Unity 開發工作。
