筆記查詢微軟產品的授權方式，讓數位應用的歷程不僅用得聰明也用得合規 😎

說明

Microsoft License
看懂伺服器新授權模式 正確評估精省採購

SQL Server

如果是實體機安裝，就是依照核心數目計價，不受 Hyper Threading 影響；虛擬機安裝 則是依照虛擬核心數目計價，所以虛擬機安裝會無法享受到 CPU Hyper Threading 的好處。

如果是使用無限虛擬機的話，則是買足全機器的核心數 SQL 授權（最少須購買 4 核心），且 SQL Server Edition 為 Enterprise 並且需要 Software Assurance；

如果是在 16 核心的實體機，啟用 6 Cores VM 並於 VM 安裝 SQL Server，則需買 6 Cores 的 SQL Server授權。

以核心數目計價，可以在同一 OSE 上可以安裝多個 SQL Server Instance。

Running Instances for Standard Edition

For each Server License, software may be run in only one Physical OSE or Virtual OSE at a time, but Customer may use any number of Running Instances of the server software in that OSE.

https://www.microsoft.com/licensing/terms/productoffering/SQLServer/OVOVS
