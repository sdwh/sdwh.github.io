SQL Server

關於授權初步的理解為：

如果是實體機安裝，就是依照核心數目計價，不受Hyper Threading影響；

如果是使用無限虛擬機的話，則是買足全機器的核心數 SQL 授權（最少須購買 4 核心），且 SQL Server Edition 為 Enterprise 並且需要 Software Assurance；

如果是在 16 核心的實體機，啟用 6 Cores VM 並於 VM 安裝 SQL Server，則需買 6 Cores 的 SQL Server授權。
