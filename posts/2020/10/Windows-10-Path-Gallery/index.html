Windows 10 資料夾路徑大全 (IIS Folder)

  1. 說明
    1. IIS 伺服器層級設定檔
    2. 系統音效 Sound Files
    3. Custom Host File to set mappings of IP addresses to host name (DNS)
    4. 快速目錄 %DirPath%
    5. Visual Studio 授權驗證資料夾

廣蒐 Windows 10 作業系統中各式重要目錄路徑方便查閱 🔍

說明

IIS 伺服器層級設定檔

%systemroot%\system32\inetsrv\config\applicationHost.config

系統音效 Sound Files

C:\windows\media

Custom Host File to set mappings of IP addresses to host name (DNS)

C:\Windows\System32\drivers\etc

快速目錄 %DirPath%

%userprofile%
%appdata%

Visual Studio 授權驗證資料夾

%LocalAppData%\Microsoft\VSCommon\OnlineLicensing
