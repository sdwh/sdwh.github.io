Chrome

Chrome 糟糕，顯示此網頁時發生問題

  1. 嘗試使用 -no-sandbox 開啟
  2. 清除快取與歷史紀錄
  3. 快取位置：%appdata%..\Local\Google\Chrome\User Data\Default\Cache

Windows

CMD 執行 python 時跳出 windows installer

  1. 使用 管理應用程式執行別名
  2. 停用 installer 相關

以不同的身分執行 SQL Server Management Studio 進行 Windows 驗證 (runas)

runas /user:domain\username "C:\Program Files (x86)\Microsoft SQL Server Management Studio 18\Common7\IDE\Ssms.exe"

更簡便的方式，在工作列上對著 SSMS Shift + 右鍵

Microsoft Docs - runas

處理 Windows 英文輸入切換的痛點

# 處理切換英文輸入的困擾
# https://blog.miniasp.com/post/2020/03/19/Devs-must-setup-multi-language-input-method

$UserLanguageList = New-WinUserLanguageList -Language "zh-TW"
$UserLanguageList.Add("en-US")
Set-WinUserLanguageList -LanguageList $UserLanguageList -Force
Set-WinDefaultInputMethodOverride -InputTip "0409:00000409"

開發者必學的多語言輸入法設定技巧：使用 Windows PowerShell 快速建立
