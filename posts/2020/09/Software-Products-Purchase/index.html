軟體採購 | 如何查詢政府軟體共同供應契約價格

  1. 說明
    1. 資訊服務採購網
    2. 經濟部工業局軟體採購辦公室

如果要為組織採購軟體該如何進行？儘管對於軟體工程師而言，技能樹理當不會點到 infra, license 等，遑論軟體採購。但實務上理解軟體採購的方式，也有助於向組織回饋意見，讓資訊環境的健全不僅是少數人的決策，可以是由下而上的從開發者的面向提供建議。本次的筆記在整理如何查詢政府軟體共同供應契約的價格。

說明

要採購軟體最便捷的方式當然是找業務報價，但如果只是在規劃階段可以藉由參考政府共同供應契約的價格，心中先有一個預算大概的藍圖，不論是作為評估或者與業務談判時的預備知識都是行的方式。

而要進行價格的查詢，需要利用到兩個網站：

資訊服務採購網

這裡可以藉由的方式搜尋到想要採購的軟體，例如 SQL Server、Windows Server 及 Visaul Studio 等。而問題是除非有政府電子採購網的帳號，否則按下我要購買後無法看到價格。這對於要做評估而言非常不方便。

好在只要根據資訊服務採購網查詢到的結果，記下標案名稱、組別及項次，前往經濟部工業局軟體採購辦公室網站進行查詢即可。

這邊以「SQL Server 企業版 2 Core 最新授權版」為例：

標案名稱：109年第一次電腦軟體共同供應契約採購
組　　別：1
項　　次：84

經濟部工業局軟體採購辦公室

進入網站後選擇 採購資訊 ，並點選 歷年標案

📝 在採購資訊的部分，如果對於共同供應契約是如何完成的也可以在此找到相關說明，例如：

接著繼續回到查價的部分，到了歷年標案後，根據要搜尋的產品選擇對應財務規格書或規格清單。這邊要注意的是從資訊服務採購網找到的標案名稱不見得都對應到最新年度規格書。例如「SQL Server 標準版 2 Core 最新授權版 (含軟體保證) GOV」對應的就是「108年第四次電腦軟體共同供應契約採購」而非「109年第一次電腦軟體共同供應契約採購」。

開啟規格書後，根據組別及項次就可以找到對應的產品價格，這邊要注意的是要參考的是「契約價格(含作業服務費)」而非「決標價」，因為經過比較後實際業務在根據共同供應契約，向政府部門提供的就是**契約價格(含作業服務費)**。

以上就是價格的查詢步驟，本次介紹了軟體的部分，規劃延伸的可以再討論的方向：

  1. 個人電腦、伺服器的價格查詢 台灣銀行 - 資訊設備共同供應契約
  2. Windows Server 及 SQL Server Cores 授權計算方式
  3. 模擬評估：維護一個機房需要多少的資訊預算
