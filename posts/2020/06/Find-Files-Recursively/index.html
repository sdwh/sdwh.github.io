如何使用指令走訪資料夾路徑下特定檔案

  1. CMD
  2. PowerShell
  3. Linux Like ls
  4. Python

各種使用指令 (Command & Sscripts) 走訪資料夾路徑下特定檔案的方式。

CMD

pushd \\UNC
dir *.js /b /s

PowerShell

Get-ChildItem -Path "\\UNC" -Filter *.js  -Recurse 
  | where {$_.CreationTime -ge "06/15/2020"} 
  | % { Write-Host $_.FullName }

Linux Like ls

ls **/**/*.js -R
