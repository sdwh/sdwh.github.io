說明

SQL Server Management Studio, SSMS

資料庫連線的管理工具，包含資料庫設計、查詢以及管理都可以藉由 SSMS 完成，最全面也是最泛用的 SQL Server 工具。

SQL Server Configuration Manager

提供 DBA 或者 IT 進行 SQL Server 伺服器組態相關設定的管理工具，包含服務的啟用、中止以及網路協定等設定。

從組態管理員管理 SQL Server Services

SQLCMD

使用 Command line 的方式與資料庫進行互動，通常用於 automation scripts 或者 trouble shooting 用途

Bulk Copy Program, BCP

大量資料存取寫入資料庫時使用的 Command line tool。

Data Migration Assistant, DMA

協助評估資料庫移轉或升級，相容性以及效能上的改善建議工具。

MSDN - DMA

SQL Server Migration Assistant, SSMA

協助評估從異質資料庫移轉至 SQL Server 的協助工作。

MSDN - SSMA

SQL Server Data Tools, SSDT

結合 Visual Studio 整合於 IDE 中進行資料庫設計、資料庫應用設計的工具。

Database Tunning Advisor, DTA

進行 SQL Server Performance Tunning 的協助工具。

MSDN - DTA

SQL Server Profiler

類似 Wireshark 可以擷取與資料庫進行中的封包行為，可用於進行 SQL Server Performance Tunning 的協助工具。
