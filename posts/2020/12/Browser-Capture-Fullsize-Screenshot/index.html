使用網頁瀏覽器進行網頁全頁面截圖 (Capture Fullsize Screenshot with Browser)

  1. 說明
    1. Edge 網頁擷取

筆記如何使用 Developer Tools 方便且迅速地將瀏覽中的網頁全頁面截圖 😎

說明

使用 Chrome 或者 Edge (Chromium Based) ，F12 啟用開發人員工具後 Ctrl + Shfit + P 啟動命令輸入介面，輸入 CAPTURE FULL SIZE SCREENSHOT 後執行即可於 Donwloads folder 取得截圖。

因為是使用 Browser 將渲染結果進行截圖，所以可能因為 CSS 或者 jQuery 語法的關係讓所見與下載有部分的差異。

Edge 網頁擷取

如果想要讓所得更貼近所見，可以考慮改使用 Edge 的網頁擷取 Ctrl + Shfit + S，藉由自行拖曳要擷取的範圍後會將截圖存於記憶體，使用小畫家或其他工具取得記憶體內容後另存成圖片即可。
